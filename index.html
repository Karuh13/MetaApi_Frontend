MetaApi

MetaApi is a project made by students of the bootcamp FullStack Developer of Upgrade Hub. The goal was to make a funcional backend with a server connected to a database and a frontend with the tools to access the different endpoints.

The MetaApi itself is a free json api that gathers the files, libraries and commands required to make an api.
